Este repositorio contiene la documentación y evidencias de las pruebas realizadas al sistema web de gestión de alumnos y calificaciones.

El objetivo de las pruebas es verificar que el sistema funciona correctamente en sus diferentes módulos, incluyendo el manejo de formularios, servicios JSON, base de datos y consumo de un servicio externo para la generación de avatares.

Conclusión
Las pruebas realizadas muestran que el sistema funciona correctamente, cumpliendo con los requisitos establecidos.
Se verificó el correcto funcionamiento de las operaciones CRUD, la validación de datos, la respuesta en formato JSON y la integración de un servicio externo.
